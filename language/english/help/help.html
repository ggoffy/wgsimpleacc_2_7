wgSimpleAcc – User Guide & Tutorial

wgSimpleAcc is a simple accounting module for XOOPS CMS designed to manage incomes, expenses, accounts, and financial reporting in a structured and user-friendly way. This guide explains installation, configuration, and daily usage.

1. Installation

Step 1: Upload the Module

  • Download the module from the GitHub repository.
  • Extract the archive.
  • Upload the folder wgsimpleacc to your XOOPS /modules/ directory.

Step 2: Install via XOOPS Admin

  • Log in to XOOPS Admin Panel.
  • Go to System → Modules.
  • Find wgSimpleAcc and click Install.
  • Confirm installation and wait for the process to complete.

2. Initial Configuration

After installation, configure the module before adding financial data.

  • Go to wgSimpleAcc → Preferences
  • Set default currency
  • Configure tax settings (if needed)
  • Enable or disable features such as departments, clients, or assets depending on your needs

3. User Roles & Permissions

wgSimpleAcc includes a flexible permission system to control access to financial operations.

Permission Types

  • View Permission – Allows users to view financial records
  • Submit Permission – Allows users to create income/expense entries
  • Approval Permission – Allows users to approve or reject entries

Setting Permissions

  • Go to System → Groups
  • Edit a group (e.g., Registered Users, Admins)
  • Assign module permissions under wgSimpleAcc

4. Core Concepts

Accounts

Accounts represent financial containers such as bank accounts, cash registers, or wallets. All transactions are linked to an account.

Income & Expenses

  • Income – Money received (donations, sales, fees)
  • Expenses – Money spent (bills, purchases, services)

Departments / Assignments

Used to categorize transactions by internal structure such as departments, projects, or activities.

Clients

Optional feature to link transactions to individuals or organizations.

Taxes

Define tax rates for automatic calculation in financial entries.

Currencies

Supports multi-currency setup depending on configuration.

5. Adding Financial Records

Adding Income

  1. Go to Income → Add New
  2. Select account
  3. Enter amount
  4. Select category/department
  5. Add optional client and tax information
  6. Save entry

Adding Expense

  1. Go to Expenses → Add New
  2. Select account
  3. Enter amount and description
  4. Select category/department
  5. Attach documents if needed
  6. Save entry

6. Approval Workflow

If approval is enabled, submitted entries must be reviewed.

  • Users submit entries (pending status)
  • Approvers review details
  • Approve or reject transactions
  • Only approved entries appear in official reports

7. Reports & Dashboard

wgSimpleAcc provides financial insights through dashboards and reports.

  • Account balance overview
  • Income vs expense charts (Chart.js)
  • Department-based summaries
  • Time-based financial reports

8. File Attachments

You can attach documents such as invoices, receipts, or images to transactions.

  • Supported file types depend on XOOPS settings
  • Files are stored securely within module directories

9. Custom Templates

The module supports template overrides for developers.

  • Modify Smarty templates inside the module
  • Create custom layouts for reports and dashboards
  • Integrate with your XOOPS theme design

10. Best Practices

  • Always define accounts before entering transactions
  • Use departments for clearer reporting
  • Enable approval workflow for financial control
  • Regularly export or backup financial data
  • Restrict permissions based on user roles

11. Troubleshooting

  • Module not visible: Ensure it is installed and active in XOOPS modules
  • No permission access: Check group permissions in System → Groups
  • Missing charts: Ensure Chart.js is loaded in your theme
  • Upload issues: Verify XOOPS upload directory permissions
